# 🚀 CodeLab NEXUS

O **NEXUS** é o meu laboratório de desenvolvimento web. O projeto utiliza uma arquitetura de **Template Fixo**, onde a identidade visual permanece consistente enquanto o núcleo central evolui com novos exercícios.

---

## 🏗️ Arquitetura do Projeto

O sistema foi projetado para ser modular, permitindo o desenvolvimento ágil de novos componentes sem a necessidade de reescrever a base do site:

* **Cabeçalho e Hero (Fixo):** Identidade visual e branding do CodeLab.
* **Viewport Central (Dinâmico):** Área dedicada ao desenvolvimento e renderização dos exercícios de HTML/CSS.
* **Footer (Fixo):** Rodapé técnico com informações de versão e créditos.

---

## 🛠️ Tecnologias & Conceitos Aplicados

* **HTML5 Semântico:** Uso de tags estruturais para separação de áreas fixas e dinâmicas.
* **CSS Master Sheet:** Estilização centralizada com variáveis (:root) para garantir consistência de cores.
* **Componentização:** Criação de classes reutilizáveis (como `.card-content`) para os exercícios.

---

## 📂 Organização de Conteúdo

Cada novo exercício ou desafio é implementado dentro da seção `<main>`, servindo como um "módulo de estudo":

- **Módulo Home:** Índice cinza de navegação.
- **Módulo Visual:** Experimentos de cores e contraste (Áreas Verde e Amarela).
