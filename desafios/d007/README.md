# 📸 Desafio 007: Imagens Dinâmicas e Flexíveis

Este projeto faz parte do meu laboratório de estudos de **HTML5** e **CSS3**. O objetivo principal foi aplicar técnicas de **Design Responsivo** e otimização de performance usando imagens que se adaptam ao tamanho da tela do usuário.

## 🚀 O que eu pratiquei neste exercício:

* **HTML5 Semântico:** Uso de tags como `<header>`, `<main>`, `<article>` e `<footer>`.
* **Imagens Responsivas:** Implementação de `srcset` e `sizes` para carregar diferentes resoluções (480px, 800px, 1200px) dependendo do dispositivo.
* **Mobile-First:** Estilização focada primeiro em celulares e expandida para desktop via `@media queries`.
* **Variáveis CSS:** Uso de `:root` para criar um Design System com cores e raios de borda padronizados.
* **SEO Básico:** Configuração de meta tags de descrição e palavras-chave.

## 🛠️ Tecnologias
* HTML5
* CSS3 (Flexbox e Variáveis)

## 📁 Estrutura de Pastas
* `/imagen`: Armazena as versões P, M e G das paisagens.
* `/css`: Contém o arquivo `style.css` com a lógica visual.
* `/icone`: Favicon do projeto.
